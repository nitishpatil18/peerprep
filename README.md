# peerprep

peer-paired mock interview platform. find matched practice partners by skill, topic, and timezone. live video + collaborative code editor.

## stack

mern (mongodb, express, react, node), socket.io, webrtc, yjs, monaco editor, redis, tailwindcss.

## structure

- `client/` react vite frontend
- `server/` express api + socket.io

work in progress.
